📌 Project Purpose
Tile Gallery is a modern web application designed to help users browse, filter, and view detailed information about various types of tiles. The primary goal of this project is to simplify the tile selection process for interior design and construction. It provides users with an intuitive and professional interface to discover the perfect tiles for their needs while ensuring a smooth, high-quality user experience.

✨ Key Features
Secure Authentication: Integrated with Better Auth to provide a robust and secure login and signup system.

Dynamic Tile Showcase: Displays a wide variety of tiles using dynamic data fetching.

Active Navigation: A smart navigation bar that highlights the current page the user is visiting.

Password Visibility Toggle: Includes an 'Eye Icon' feature on the signup page to let users show or hide their password.

Premium News Ticker: A modern, high-speed Marquee on the home page for displaying the latest updates and announcements.

Protected Routes: Middleware implementation to prevent unauthorized access to private pages like the user profile.

Responsive & Minimalist Design: Built with HeroUI and Tailwind CSS to ensure a clean, professional look across all devices.
